2018 Code

To view our robot code, type the following into the command line:

$ cd #enter your desired directory
$ git clone http://github.com/HarkerRobo/RoboCode2018
$ cd RobotCode2018
$ git fetch origin
$ git checkout master

To view our scouting app, type the following into the command line:

$ cd #enter your desired directory

#for Android:
$ git clone http://github.com/HarkerRobo/scouting-android
$ cd scouting-android

#for iOS:
$ git clone http://github.com/HarkerRobo/scouting-ios
$ cd scouting-ios

$ git fetch origin
$ git checkout master
