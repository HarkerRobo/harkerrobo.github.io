Welcome to Harker Robotics.

This is the landing page for Harker's FRC Robotics Team 1072.

If you want to view our robot code for the 2014-2015 Build Season, type

$ cd harkerrobo/RobotCode2015
$ git fetch origin
$ git checkout master

in the command line.

Or, if you would like to view our image processing code for the 2014-2015 Build Season, type

$ cd harkerrobo/ImageProcessing2015
$ git fetch origin
$ git checkout master
